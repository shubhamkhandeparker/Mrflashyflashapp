# 🔦 MrFlashyFlashApp

**MrFlashyFlashApp** is a minimal, lightweight Android flashlight application built using **Kotlin** and **Jetpack Compose**. It leverages the **Camera2 API** to toggle the flashlight (torch) on and off using a simple and elegant UI.

---

## 🚀 Features

- ✅ Toggle flashlight ON/OFF using a single button
- ✅ Real-time torch state updates
- ✅ Elegant and clean UI using Jetpack Compose
- ✅ Efficient use of Android's `CameraManager` for torch control
- ✅ Logs and handles common exceptions gracefully

---

## 🛠️ Tech Stack

| Technology | Description                            |
|------------|----------------------------------------|
| Kotlin     | Primary programming language           |
| Jetpack Compose | Modern UI toolkit for Android      |
| Camera2 API | Controls the flashlight (torch)       |
| MVVM (Lite) | Separation of concerns and state mgmt |

## 📦 How to Run

1. Clone the repository:

```bash
git clone https://github.com/shubhamkhandeparker/Mrflashyflashapp.git
````

2. Open in **Android Studio**.

3. Connect your Android device or start an emulator.

4. Click **Run** ▶️.
